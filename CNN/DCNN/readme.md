# 深度卷积神经网络
    在小型化方面常用的手段有：
    （1）卷积核分解，使用1×N和N×1的卷积核代替N×N的卷积核
    （2）使用bottleneck结构，以SqueezeNet为代表
    （3）以低精度浮点数保存，例如Deep Compression
    （4）冗余卷积核剪枝及哈弗曼编码
